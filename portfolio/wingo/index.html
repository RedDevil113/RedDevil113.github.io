Принципы Wingo Plan

Наглядность

В любой момент момент, запустив Wingo Plan, вы видите какие проекты или дела идут по плану, а по какием есть риски выбиться из графика и возможны проблемы проблемы

Иерархия

Каждый проект делится на этапы, этапы - на задачи.

Устанавливайте сроки для промежуточных этапов, определяйте список задач, необходимых для его успешного выполнения

Важность

Часто на задачу нужно немного времени, но она критически важна для успеха этапа и проекта в целом и наоборот.

Устанавливайте Важность для каждой задачи, исходя их того, насколько она влияет на успехное завершение этапа.
